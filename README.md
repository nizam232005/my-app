# Remainder App 📝

A simple and clean **Reminder App** built with React. Add your reminders, view them in a list, and delete them when done — all in the browser with no backend required.

-----

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```bash
git clone https://github.com/your-username/my-app.git
cd my-app
npm install
npm start
```

The app will open at **<http://localhost:3000>**

-----

## 📋 Features

- Add reminders via a text input
- View all reminders in a clean list
- Delete individual reminders
- Empty state message when no reminders exist
